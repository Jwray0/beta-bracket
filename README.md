# beta-bracket
# D12N52 Hall Effect Magnet Mount & Bracket

A **3D-printable magnet mount and bracket system** designed for **D12 N52 neodymium magnets**, intended for use in **Hall effect–based controllers and analog input mechanisms**.

This project focuses on **precise magnet alignment**, **repeatable magnet-to-sensor spacing**, and **mechanical stability**, making it suitable for high-precision controller mods and custom input hardware.

---

## ✨ Features

* Designed specifically for **Ø12 mm N52 neodymium magnets (D12N52)**
* Optimized for **Hall effect sensors** used in controllers
* Maintains consistent and repeatable **magnet-to-sensor spacing**
* Prevents unwanted magnet rotation, drift, or axial misalignment
* Fully **3D printable** on common FDM printers
* No specialized hardware required

---

## 🕹️ Intended Use Cases

* Hall effect joystick assemblies
* Custom game controllers
* Controller mods and replacements for potentiometer-based designs
* Experimental or prototyping input devices

This design is intended to help replace or augment traditional potentiometer systems with **contactless magnetic sensing** for improved longevity and consistency.

---

## 🧲 Magnet & Sensor Notes

### Magnet Specification

* **Type:** Neodymium
* **Size:** Ø12 mm
* **Grade:** N52

* **D12 N52 magnets can be purchased from K&J Magnetics** https://www.kjmagnetics.com/d12-n52-neodymium-cylinder-magnet

Using a different magnet size or grade will affect sensor output and may require design changes.


---

## 🖨️ 3D Printing Guidelines

### Recommended Materials

* **PLA/PLA-CF** – easiest to print, sufficient for most controller use
* **PETG** – better heat resistance and durability
* **ABS/ABS-GF** – optional, requires enclosure


### Print Settings (Suggested)

* Layer height: 0.2 mm
* Infill: Default
* Walls: Default
* Supports: Not required 

### Tolerances

* Designed for **press-fit**
* Light sanding may be required depending on printer calibration

---

## 🔧 Assembly

1. Verify magnet polarity before installation
2. Press-fit the D12 N52 magnet into the mount
3. Ensure the magnet mount is seated fully and squarely to Stick box arm
4. Mount the bracket relative to the Stick box covering the mount
5. Verify Bracket if fully seated and flush
