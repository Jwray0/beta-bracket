# beta-bracket
Hall Effect Magnet Mount &amp; Bracket
This project provides a 3D-printable magnet mount and bracket system designed specifically for D12 N52 neodymium magnets, intended for use in Hall effect–based controllers and input devices.

The design focuses on precise magnet alignment, repeatable sensor spacing, and mechanical stability, ensuring consistent and predictable Hall sensor readings. It is ideal for custom controllers, joystick mods, trigger assemblies, and other analog input mechanisms that rely on magnetic sensing rather than potentiometers.

Features

Designed for D12 N52 neodymium magnets

Optimized for Hall effect sensors

Maintains consistent magnet-to-sensor spacing

Rigid bracket design to prevent magnet drift or rotation

Fully 3D printable with no specialized hardware required

Modular and easy to adapt for custom controller layouts

Use Cases

Hall effect joystick or trigger assemblies

Custom game controllers and mods

Precision analog input devices

Prototyping and experimental input hardware

Printing Notes

Designed for common FDM printers

PLA, PETG, or ABS recommended

Orientation and tolerance optimized for press-fit or adhesive mounting

This project is intended to be open, hackable, and mod-friendly. Feel free to remix, improve, or adapt it for your own controller designs.
